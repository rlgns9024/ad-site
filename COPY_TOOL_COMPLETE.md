# 🎉 복사 도구(copy-tool.html) 완성!

벤치마킹 사이트와 동일한 복사 도구 페이지를 완성했습니다!

---

## ✅ 생성된 파일

### 1. **copy-tool.html**
- 완전한 HTML 페이지
- 메인 사이트와 자연스럽게 연결
- 공통 헤더/푸터 포함
- 라우팅 완료

### 2. **assets/css/copy-tool.css**
- 사이드바 카테고리 스타일
- 이모지 그리드 레이아웃
- 커스텀 입력 영역
- 반응형 디자인
- 다크 모드 지원

### 3. **assets/js/copy-tool.js**
- 6가지 카테고리 (감정, 음식, MBTI, 직업, 여행, 축하)
- 이모지 선택 및 추가
- 커스텀 조합 기능
- 복사 및 저장 기능
- localStorage 연동

---

## 📋 주요 기능

### 좌측 사이드바 카테고리
```
😊 감정·표현
🍔 음식·음료
💅 MBTI별
💼 직업·일무
✈️ 여행·종기
🎉 축하·기념일
```

### 중앙 이모지 그리드
- 카테고리별 이모지 표시
- 클릭으로 커스텀에 추가
- Hover 효과
- 반응형 배치

### 하단 커스텀 섹션
```
✏️ 나만의 이모지 조합 만들기

[입력창] [초기화]

[📋 조합 복사] [💾 저장하기]

자주 쓰는 이모지: ❤️ 😊 🔥 ...
```

---

## 🎯 기능 상세

### 1️⃣ 카테고리 선택
- 버튼 클릭 시 해당 카테고리 이모지 표시
- Active 상태 강조
- 부드러운 전환 애니메이션

### 2️⃣ 이모지 추가
- 그리드의 이모지 클릭 시 커스텀 입력에 추가
- 시각적 피드백
- 순차적 추가 가능

### 3️⃣ 입력 영역
- 직접 이모지 입력 가능
- 실시간 업데이트
- 초기화 버튼

### 4️⃣ 복사 기능
- `navigator.clipboard` 사용
- 복사 후 Toast 메시지
- Fallback 지원

### 5️⃣ 저장 기능
- localStorage에 저장
- 중복 체크
- 피드백 메시지

### 6️⃣ 빠른 이모지
- 자주 쓰는 이모지 12개
- 한 클릭으로 추가
- 빠른 조합 생성

---

## 📱 반응형 지원

### 데스크톱 (> 1024px)
- 사이드바 250px (고정)
- 이모지 그리드: 4~6열
- 완벽한 레이아웃

### 태블릿 (768px ~ 1024px)
- 사이드바 200px
- 이모지 그리드: 3~4열
- 최적화된 배치

### 모바일 (≤ 768px)
- 사이드바: 가로 스크롤
- 이모지 그리드: 2~3열
- 터치 친화적
